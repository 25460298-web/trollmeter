La app Troll-O-Meter 3000 es una herramienta hecha en Streamlit que detecta qué tan tóxico o amigable es un mensaje usando un modelo Naive Bayes simplificado que tú mismo entrenas.
Permite:
Entrenar a la IA con frases marcadas como tóxicas o pro.
Analizar nuevos mensajes y mostrar el nivel de toxicidad.
Explicar qué palabras influyeron en la decisión.
